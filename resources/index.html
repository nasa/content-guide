Style guides

We generally follow AP style unless otherwise noted. If there is a difference in addressing an item between the AP style and NASA style, use the NASA style. If there is a difference in addressing an item between the NASA Stylebook or Graphics Standards and this guide, use the Stylebook or Graphics Standards.
